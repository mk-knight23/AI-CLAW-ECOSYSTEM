## 🚀 Deployment & Usage

### 1. Local Exploration
Open `index.html` in the root directory to enter the **Genesis Portal**, an interactive 3D-parallax experience narrating the lineage of the ecosystem.

### 2. GitHub Pages Deployment
This ecosystem is optimized for GitHub Pages. Simply:
1. Push this repository to GitHub.
2. Enable **Settings > Pages > Deploy from branch (main)**.
3. Access your ecosystem at `your-username.github.io/claw-ecosystem/`.
